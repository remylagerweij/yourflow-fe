# HomeInteractiveCases

Interactive carousel showcasing client case studies. Adapts layout for desktop (vertical) and mobile (horizontal).

## Usage

```vue
<HomeInteractiveCases @ready="onReady" />
```

## Props

None.

## Events

| Event | Payload | Description                                     |
| ----- | ------- | ----------------------------------------------- |
| ready | -       | Emitted when the Swiper instance is initialized |

## Slots

None.

## Examples

### Basic

```vue
<HomeInteractiveCases />
```
