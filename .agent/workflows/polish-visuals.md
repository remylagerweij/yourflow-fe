# Polish Visuals Workflow

This workflow uses **UX Design Skill** sections: Visual Design + Wow Factor.

> [!TIP]
> Use `view_file` to read `.agent/skills/ux-design/SKILL.md` sections 1 (Visual Design) and 5 (Wow Factor).
> Use `view_file` to read `.agent/skills/ui-ux-pro-max/SKILL.md`

---

## Steps

1.  **Identify Target**
    - What component/page needs visual polish?

2.  **Visual Audit**
    - [ ] Consistent colors/spacing/typography?
    - [ ] Clear visual hierarchy?
    - [ ] Adequate whitespace?
    - [ ] Dark mode looks premium?

3.  **Wow Factor Check**
    - [ ] Gradients instead of flat colors?
    - [ ] Multi-layer shadows for depth?
    - [ ] Glassmorphism where appropriate?
    - [ ] Premium typography (large headlines, refined labels)?

4.  **Apply Improvements**
    - Use patterns from UX Design Skill section 5.
    - Focus on high-impact, low-effort wins first.

## Quick Wins

```css
/* Instant upgrade: layered shadow */
box-shadow:
  0 1px 2px rgba(0, 0, 0, 0.05),
  0 4px 8px rgba(0, 0, 0, 0.1);

/* Instant upgrade: gradient accent */
background: linear-gradient(135deg, #667eea, #764ba2);
```
